## How to run a Bot on Termux :

**STEP 1**
```
pkg update && pkg upgrade && pkg install bash && pkg install git && git clone https://github.com/RasZRengokai/Shizukav1.1-edt-rasz
```
**STEP 2**
```
cd Shizukav1.1 
```
**STEP 3**
```
chmod 755 run.sh && ./run.sh
```
